# E-Commerce Django Web Application

A full-featured e-commerce web application built with Django, HTML, CSS and JavaScript.  
Supports user authentication, category-wise product listing, shopping cart, wishlist, and a clean responsive UI (no Bootstrap).

## üéØ Features

- User registration, login and logout  
- Category-wise product listing and filtering  
- Product detail pages with full information  
- Shopping cart: add items, edit quantities, remove items  
- Wishlist: add items, move to cart or remove  
- Clean, responsive UI built with pure CSS (no Bootstrap)  
- Image uploads for products (with media handling)  
- Admin interface for managing categories and products  
- Simple quantity controls (increase / decrease)  
- Protective access: cart & wishlist only accessible by logged-in users  
- Modern look and feel with object-fit images, card layout, and minimal design  

## üìã Features

- **Log in or register** a new user to start using the website.  
- **Admin panel:** Go to `/admin/` and add **categories** and **products** (supports image uploads).  
- **Homepage:** Browse all products or **filter by category**.  
- **Product page:** View product details and **add items to Cart or Wishlist**.  
- **Cart Page:**  
  - Increase/decrease item quantity  
  - Remove items  
  - View **grand total**
- **Wishlist Page:**  
  - Move items to cart  
  - Remove items
- **Access Control:**  
  - Only **logged-in users** can view Cart and Wishlist.
